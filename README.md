# Corona-Detection-Project
Project Title: Corona Detection and Management System

Overview:
This is a complete desktop application developed in Python that facilitates COVID-19 risk assessment and patient management. The system allows healthcare providers to register patients, conduct corona symptom evaluations, automatically calculate risk scores, and store test results in a secure database.

Key Features:

📝 Patient registration and profile management

🔍 Symptom-based corona risk assessment

📊 Automated test result classification (Positive/Negative)

⚠️ Severity level determination (Low/Medium/High)

📈 Analytical dashboard with visual charts

💾 Secure SQLite database storage

🖥️ User-friendly Tkinter GUI interface

Technical Stack:

Frontend: Tkinter (Python)

Backend: Python

Database: SQLite

Visualization: Matplotlib

Impact: Helps healthcare facilities efficiently manage COVID-19 screening, maintain organized patient records, and generate valuable insights through data analytics.
